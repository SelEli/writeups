## Récupération d'accès Windows via dump SAM — Kali Linux

Diagnostic et résolution autonome d'un problème de perte
d'accès domaine sur 4 postes Windows en environnement
professionnel réel. Boot Kali live, accès SAM, suppression
des hash, restauration des credentials réseau d'origine.

Compétences : live boot, manipulation SAM, jonction domaine
Windows, intervention en conditions réelles.

## Approche générale

Résolution de problèmes réels par compréhension des systèmes
plutôt que par application de procédures. Raisonnement
orienté architecture et surface d'attaque. Documentation
systématique des méthodes et apprentissages.
